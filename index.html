Introduction

The following is the latest results for the current work.

Description

First, in our features we followed Cross-Domain Polarity Classification on Twitter [1] to specify the polarity for tweets, we created four different tweet representations and trained one classifier on each one of them, trying to find the parameters that achieve the highest accuracy (Text-Based Representation, Feature-Based Representation, Lexicon-Based Representation, Combined Representation).

We have separated our classifiers in two categories, based on their domain-(in)dependent nature: the hybrid classifier (HC) and the lexicon-based one (LC) .

How To Run ?

// Training.
InstanceList trainingInstances = IOUtils.readDataset('training.csv');
TerrorismTendencyClassifier tendencyClassifier = new TerrorismTendencyClassifier();
tendencyClassifier.train(trainingInstances);
tendencyClassifier.saveModel();

// Testing.
InstanceList testingInstances = IOUtils.readDataset('testing.csv');
TerrorismTendencyClassifier tendencyClassifier = new TerrorismTendencyClassifier();
tendencyClassifier.loadPreviousClassifier('classifier.bin');
tendencyClassifier.predict(testingInstances);
